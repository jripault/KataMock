Et mon test unitaire ?

  • On teste une méthode d'une classe.
  • La classe peut avoir des dépendances.
  • Le test doit être déterministe.
  • Ainsi les dépendances doivent être contrôlées.
  • Exemple : un test unitaire ne doit pas accèder au file system, une base de données, le réseau. Ce sont des éléments variables.
